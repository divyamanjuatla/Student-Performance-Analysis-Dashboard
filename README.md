# Student-Performance-Analysis-Dashboard
Student Performance Analysis using Python, Pandas, Matplotlib, and Seaborn with Exploratory Data Analysis (EDA) and interactive visualizations.

##  Project Overview

This project analyzes student academic performance using Python and Exploratory Data Analysis (EDA). The objective is to identify how various factors such as Gender, Parent Education, Weekly Study Hours, Test Preparation, Sports Participation, and Parent Marital Status influence students' Math, Reading, and Writing scores.

The project demonstrates the complete data analysis workflow, including data cleaning, handling missing values, exploratory data analysis, visualization, and extracting meaningful insights.

---

##  Objectives

- Clean and preprocess the dataset.
- Handle missing values using appropriate statistical methods.
- Analyze the impact of demographic and educational factors on student performance.
- Compare average scores across different student groups.
- Visualize patterns using professional charts and heatmaps.
- Generate meaningful insights for educational decision-making.

##  Key Insights

- Male students achieved higher average scores in Mathematics.
- Female students performed better in Reading and Writing.
- Students with higher weekly study hours generally achieved better scores.
- Test preparation positively impacted performance across all subjects.
- Students whose parents had higher educational qualifications generally performed better.
- Reading and Writing scores showed a strong positive correlation.
